# Card

Card component is a standlone component that is often used inside parent component card-group. It displays heading, paragraph, button and optional image. By default width of card is 100% width. Once placed in card group, each card will be set to proper width.

## APIs
**`headline`: string**: Headline text to be added at top of card.

**`buttontext`: string** Button text inside card component. Button has brand styling by default and cannot be changed. 

**`text`: string** Main paragraph text to be added inside card component.

**`src`: string** (optional) Image src to be added.

Example usage:
```
<c360-card
  headline="About us" 
  buttontext="Click here"
  text="This is a paragraph describing the card information. Recommendation length is 350 char."
  src="../image.jpg"
/>
```
