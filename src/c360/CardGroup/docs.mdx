# Card Group

Card Group component is a parent component that contain card components.
It sets the grid for the card components and lays them out in proper layout/spacing for all breakpoints.

## APIs
**`card`: data** Sets number of card component in

**`cardgroupheading`: string** (Optional) Heading for the card group.

Example usage:
```
<c360-card-group 
  cardgroupheading="Want to learn more about Salesforce?" 
  card="cardItems"
/>
```
